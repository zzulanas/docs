title: "Quickstart: Create Your First 3D Scene with Splat"
description: "Upload a video, trigger GPU reconstruction, and download a web-ready 3D Gaussian Splat in under 5 minutes of setup using the Splat REST API."
---

This guide walks you through the full Splat workflow from scratch. By the end you will have created a scene, uploaded a video, triggered GPU reconstruction, polled for completion, and downloaded the resulting 3D model. The whole process takes about 5 minutes of setup, plus pipeline processing time (typically 8–15 minutes for the `standard` preset).

<Info>
  You need an API key before you start. Create one in the [Splat dashboard](https://splat-3d.com/dashboard) under **Settings > API Keys**. See the [Authentication guide](/authentication) for details.
</Info>

## Set your API key

Export your API key as an environment variable so you can reference it in the examples below.

```bash
export SPLAT_API_KEY="s3d_your_key_here"
```

<Steps>

<Step title="Create a scene">

Creating a scene registers your intent to upload and process a file. The response returns a `sceneId` you will use for all subsequent requests, and a presigned `uploadUrl` you will use to upload your file directly to cloud storage.

<CodeGroup>

```bash cURL
curl -X POST https://api.splat-3d.com/api/v1/scenes \
  -H "Authorization: Bearer $SPLAT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "My First Scene",
    "content_type": "video/mp4",
    "preset": "standard"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.splat-3d.com/api/v1/scenes",
    headers={"Authorization": f"Bearer {SPLAT_API_KEY}"},
    json={
        "title": "My First Scene",
        "content_type": "video/mp4",
        "preset": "standard",
    },
)
data = response.json()["data"]
scene_id = data["sceneId"]
upload_url = data["uploadUrl"]
```

```javascript JavaScript
const response = await fetch("https://api.splat-3d.com/api/v1/scenes", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${SPLAT_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    title: "My First Scene",
    content_type: "video/mp4",
    preset: "standard",
  }),
});
const { data } = await response.json();
const { sceneId, uploadUrl } = data;
```

</CodeGroup>

The response contains the `sceneId` and presigned `uploadUrl`:

```json
{
  "data": {
    "sceneId": "a1b2c3d4e5f6",
    "uploadUrl": "https://..."
  },
  "meta": {
    "request_id": "..."
  }
}
```

</Step>

<Step title="Upload your video">

PUT your video file directly to the presigned URL. This uploads straight to cloud storage — no data passes through the Splat API servers and there is no size limit on the API side (max file size is 2 GB).

```bash
curl -X PUT "$UPLOAD_URL" \
  -H "Content-Type: video/mp4" \
  --data-binary @walkthrough.mp4
```

<Tip>
  For best reconstruction results, record a slow, steady walkthrough with good, even lighting. Avoid fast pans and motion blur. A 30–60 second clip typically produces excellent results. The more overlap between frames, the better the camera pose estimation.
</Tip>

</Step>

<Step title="Start processing">

Once the upload completes, call the `/process` endpoint to kick off the reconstruction pipeline. Splat will run structure-from-motion to estimate camera poses, then train a 3D Gaussian Splat model on A100 GPUs.

<CodeGroup>

```bash cURL
curl -X POST https://api.splat-3d.com/api/v1/scenes/$SCENE_ID/process \
  -H "Authorization: Bearer $SPLAT_API_KEY"
```

```python Python
requests.post(
    f"https://api.splat-3d.com/api/v1/scenes/{scene_id}/process",
    headers={"Authorization": f"Bearer {SPLAT_API_KEY}"},
)
```

```javascript JavaScript
await fetch(`https://api.splat-3d.com/api/v1/scenes/${sceneId}/process`, {
  method: "POST",
  headers: { Authorization: `Bearer ${SPLAT_API_KEY}` },
});
```

</CodeGroup>

</Step>

<Step title="Poll for completion">

The pipeline runs asynchronously. Poll `GET /api/v1/scenes/{id}` until the status field returns `complete` or `failed`. For the `standard` preset, expect 7–15 minutes of processing time.

<CodeGroup>

```bash cURL
curl https://api.splat-3d.com/api/v1/scenes/$SCENE_ID \
  -H "Authorization: Bearer $SPLAT_API_KEY"
```

```python Python
import time

while True:
    response = requests.get(
        f"https://api.splat-3d.com/api/v1/scenes/{scene_id}",
        headers={"Authorization": f"Bearer {SPLAT_API_KEY}"},
    )
    status = response.json()["data"]["status"]
    print(f"Status: {status}")

    if status == "complete":
        break
    elif status == "failed":
        print("Processing failed:", response.json()["data"].get("error"))
        break

    time.sleep(15)
```

```javascript JavaScript
const poll = async () => {
  while (true) {
    const res = await fetch(
      `https://api.splat-3d.com/api/v1/scenes/${sceneId}`,
      { headers: { Authorization: `Bearer ${SPLAT_API_KEY}` } }
    );
    const { data } = await res.json();
    console.log(`Status: ${data.status}`);

    if (data.status === "complete") return data;
    if (data.status === "failed") throw new Error(data.error);

    await new Promise((r) => setTimeout(r, 15000));
  }
};
```

</CodeGroup>

While processing, the response includes the current stage and overall progress:

```json
{
  "data": {
    "status": "processing",
    "stage": "training",
    "overallPct": 65
  }
}
```

When the job finishes, the response includes a download URL and quality metrics:

```json
{
  "data": {
    "status": "complete",
    "download_url": "/api/v1/scenes/a1b2c3d4e5f6/download",
    "format": "sog",
    "metrics": {
      "psnr": 24.5,
      "num_gaussians": 1940000,
      "training_time_seconds": 180
    }
  }
}
```

</Step>

<Step title="Download the result">

Download the reconstructed 3D model. The default format is SOG — a compressed, web-optimized Gaussian Splat that loads instantly in any browser. Append `?format=ply` to get the raw PLY file for use in offline tools.

<CodeGroup>

```bash cURL
# Download SOG (web-optimized, default)
curl -O https://api.splat-3d.com/api/v1/scenes/$SCENE_ID/download \
  -H "Authorization: Bearer $SPLAT_API_KEY"

# Download PLY (raw, for offline tools)
curl -O "https://api.splat-3d.com/api/v1/scenes/$SCENE_ID/download?format=ply" \
  -H "Authorization: Bearer $SPLAT_API_KEY"
```

```python Python
# Download SOG
response = requests.get(
    f"https://api.splat-3d.com/api/v1/scenes/{scene_id}/download",
    headers={"Authorization": f"Bearer {SPLAT_API_KEY}"},
)
with open(f"scene-{scene_id}.sog", "wb") as f:
    f.write(response.content)
```

```javascript JavaScript
const res = await fetch(
  `https://api.splat-3d.com/api/v1/scenes/${sceneId}/download`,
  { headers: { Authorization: `Bearer ${SPLAT_API_KEY}` } }
);
const blob = await res.blob();
```

</CodeGroup>

</Step>

</Steps>
